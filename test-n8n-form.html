N8N Form Test

Тестовая страница для проверки интеграции форм с N8N

ℹ️ Информация о тесте

  • Webhook URL: https://n8n.lakestrom.com/webhook/contact-form
  • Метод: POST с JSON payload
  • Обязательные поля: Имя, Фамилия, Email, Город, Сообщение
  • Honeypot поле: website (должно быть пустым)
  • После отправки форма отправит уведомление в Telegram
