What Changed

  • Previous Issue: Forms were replaced with HTML, breaking Framer's functionality
  • Solution: N8N Handler V2 intercepts form submissions without replacing HTML
  • Method: Uses event capture phase to intercept before Framer

How It Works

  1. User fills out Framer form normally
  2. On submit click, V2 handler captures the event
  3. Form data is collected and sent to N8N webhook
  4. Success/error feedback shown to user
  5. Form clears on successful submission

Test Instructions

  1. Open the form below in the iframe
  2. Try typing in the form fields
  3. Fill all required fields:
    • Имя (First Name)
    • Фамилия (Last Name)
    • Email
    • Город (City)
    • Сообщение (Message)
  4. Click "Отправить" to submit
  5. Watch for success message

Console Commands
